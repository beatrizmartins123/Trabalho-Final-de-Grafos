# Monitoramento Pós‑Cesariana – Chatbot Inteligente

Chatbot para Telegram que realiza telemonitoramento de puérperas após cesariana, utilizando **PLN (spaCy)** e uma **árvore de decisão implementada como DAG** para triagem de risco de infecção de sítio cirúrgico.

## Requisitos

- Python 3.9+
- [spaCy](https://spacy.io/) com modelo `pt_core_news_sm`
- [python-telegram-bot](https://python-telegram-bot.org/) v20+
- 

## Instalação
Necessário instalar isso parar rodar em sua maquina:


pip install python-telegram-bot spacy
python -m spacy download pt_core_news_sm

Depois disso, vá para o diretório que se encontra o código fonte e rode "python Chatbot_PLN_Dag.py".
Uma vez com o chatbot rodando, você poderá acessar e conversar com o bot por esse link: https://t.me/pibicbot
